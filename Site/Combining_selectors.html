Combining Selectors

This page is a smaple that teaches how to use combinig selectors of CSS.

Following are three different types of selectors

Element Selector

Element selector is used to provide styling to some particular elements in the file.

Class Selector

Using Class Selector all the elements of the the file under the particular class could be given the same styling.

Id Selector

Id selector is used to provide styling to the element with the particular id value. This cannot be reused because multiple elements cannot share the same id value.
